## Description
*Sol de Noche* is an immersive artistic residency project focused on experimental audiovisual work and participatory performance. The project explores themes of nocturnal cycles, collective memory, urban ecology, and sensory perception. Through interdisciplinary collaboration and site-specific interventions, the work investigates how communities experience space, light, and temporality.
